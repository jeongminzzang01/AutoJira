        사용 가이드

        1. 업데이트 일정 만들기
        1+ 월별 일정 추가 → 연도/월 선택 → 해당 월의 모든 목요일이 자동 생성됩니다.
        2각 업데이트의 + 항목 추가로 작업 항목을 등록합니다.
        3Jira에서 불러오기 탭에서 기존 Jira 티켓을 검색해 가져올 수도 있습니다.
        2. 새 티켓 + 서브태스크 생성
        1+ Jira 일괄 생성 → 새 티켓 생성 탭 선택
        2Jira 미연결 항목을 체크 → 다음
        3이슈 유형(Epic/Task), 수정버전, 담당자 설정
        4서브태스크 템플릿을 확인/수정 (펼쳐서 담당자, 컴포넌트 등 설정)
        5Jira 티켓 생성 또는 CSV 다운로드
        3. 기존 에픽/태스크에 서브태스크 추가
        A방법 1 - 테이블에서 직접: Jira 연결 항목의 체크박스 선택 → 하단 하위 서브태스크 생성 클릭
        B방법 2 - 일괄 생성: + Jira 일괄 생성 → 기존 티켓 하위에 생성 탭
        !기존 하위 항목이 있으면 자동으로 표시되어 중복 생성을 방지할 수 있습니다.
        4. CSV 일괄 생성 (빠른 생성)
        1CSV 일괄 생성 클릭 → 항목 이름을 줄 단위로 붙여넣기 또는 CSV 파일 업로드
        2항목 파싱 → 티켓 설정 → Jira 티켓 생성 또는 CSV 다운로드
        서브태스크 템플릿 치환 규칙
        [전체제목] → 상위 티켓의 전체 이름 (예: [M39] 카드 시스템)
        [마일스톤] → 마일스톤 태그만 (예: [M39])
        [항목명] → 마일스톤 제외한 순수 이름 (예: 카드 시스템)

        예시: [전체제목] - 기획서 작성 → [M39] 카드 시스템 - 기획서 작성
